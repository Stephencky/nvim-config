# nvim-config

## 用法
git clone https://github.com/Stephencky/nvim-config.git ~/.config/nvim

## 说明
* 使用packer vim作为插件管理器
* core目录放置核心的插件配置
* plugins目录放置插件对应的配置
* plugins-setup.lua 文件用于新增插件，保存该文件会自动下载插件，或者执行:PackerInstall命令
